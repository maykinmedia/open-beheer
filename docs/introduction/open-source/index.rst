Open-source
===========

The openbeheer is open-source and available 
under the `EUPL license`_.

In addition, this project makes use of various open-source `Python libraries`_ 
and `npm packages`_ under the hood.

.. _`Python libraries`: https://github.com/maykinmedia/open-beheer/blob/master/requirements/base.txt
.. _`npm packages`: https://github.com/maykinmedia/open-beheer/blob/master/package-lock.json
